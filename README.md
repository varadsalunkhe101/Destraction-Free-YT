# DFYT - Distraction Free YouTube

## Overview

DFYT (Distraction Free YouTube) is a simple educational web platform designed for engineering students and self-learners. The goal is to provide a focused learning environment by organizing educational content into categories without distractions such as advertisements, comments, shorts, or unnecessary recommendations. CREATED USING ONLY HTML FOR PRACTICE PURPOSE
